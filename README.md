# LangGraph-Practice
# How to run?
### STEPS:

Clone the repository

```bash
https://github.com/Tokir-Vora/LangGraph-Practice.git
cd LangGraph-Practice
```
### STEP 01 - Create a virtual environment after opening the repository

```bash
python -m venv venv
source venv/bin/activate
```

### STEP 02 - install the requirements

```bash
pip install -r requirements.txt
```

### STEP 03 - create .env file

```bash
GROQ_API_KEY = < Provide your Groq api token >
GOOGLE_API_KEY= <Provide your Google api token>
LANGCHAIN_TRACING_V2=true
LANGCHAIN_API_KEY=<Provide your LangSmith api token>
TAVILY_API_KEY=<Provide your Tavily api token>
LANGCHAIN_PROJECT= <Provide your LangSmith project name>
```
